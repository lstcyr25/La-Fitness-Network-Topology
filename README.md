# LA Fitness Network Topology (Cisco Packet Tracer)

## 📊 Overview

This project simulates a structured enterprise network for a fitness facility using Cisco Packet Tracer.

The design was inspired by real-world observations from a current/previous workplace, modeling how devices, departments, and communication systems operate within a gym environment.

---

## 🏢 Real-World Context

This network design reflects practical insights from observing daily operations within a fitness facility, including:

* Front desk check-in systems
* Sales team workstations and communication
* Use of IP phones for internal coordination
* Shared resources such as printers and servers
* Wireless access for staff and guests

By modeling this environment, the project demonstrates the ability to apply networking concepts to realistic business scenarios.

---

## 🎯 Objectives

* Design a scalable network for a business environment
* Segment departments using VLANs
* Enable communication between networks (inter-VLAN routing)
* Simulate real-world device usage and connectivity
* Ensure reliable communication across all devices

---

## 🛠️ Tools & Technologies

* Cisco Packet Tracer
* Networking Fundamentals (TCP/IP, VLANs, Subnetting, Routing)

---

## 🖧 Network Design

### Departments Modeled

* Front Desk Network
* Sales Network

### Key Features

* VLAN segmentation for departmental separation
* Router-based inter-VLAN routing
* Integration of PCs, IP phones, printers, and server
* Wireless access point for connectivity
